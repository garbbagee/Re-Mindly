# Re-Mindly

Re-Mindly is a mobile app designed for students to manage daily tasks and reminders. Its clean and simple interface helps users stay organized and boost productivity by tracking tasks and deadlines efficiently.

## Features

- Create, edit, and delete tasks
- Persistent reminders to combat procrastination
- View history of completed tasks
- Basic customizable settings

## Technologies

- Ionic Framework with Angular
- Firebase Firestore (planned)
- Capacitor for native device features
